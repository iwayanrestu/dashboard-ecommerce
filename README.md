# E-Commerce Dashboard

Dashboard interaktif untuk menganalisis performa penjualan, perilaku pelanggan, dan efisiensi pengiriman pada dataset e-commerce.

---

## Tujuan Project

Project ini bertujuan untuk:

- Menganalisis tren penjualan
- Mengidentifikasi kategori produk dengan performa terbaik
- Mengetahui distribusi metode pembayaran
- Memahami persebaran pelanggan
- Mengevaluasi performa pengiriman

---

## Fitur Dashboard

Dashboard ini memiliki beberapa bagian utama:

- Key Metrics

  - Total Orders
  - Total Revenue
  - Average Delivery Time

- Sales Trend

  - Tren jumlah order per bulan

- Top Product Categories

  - Kategori dengan revenue tertinggi

- Payment Methods

  - Distribusi metode pembayaran

- Customer Distribution

  - Persebaran pelanggan berdasarkan wilayah

- Delivery Performance
  - Perbandingan pengiriman tepat waktu dan terlambat

---

## Insight Utama

- Kategori produk tertentu memberikan kontribusi terbesar terhadap revenue
- Metode pembayaran yang paling banyak digunakan adalah credit card
- Pelanggan terkonsentrasi di wilayah tertentu
- Sebagian besar pengiriman berada dalam kategori tepat waktu

---

## Teknologi yang Digunakan

- Python
- Pandas
- Streamlit
- Plotly

---

## Cara Menjalankan Project

1. Clone repository:
   git clone https://github.com/iwayanrestu/dashboard-ecommerce.git

2. Masuk ke folder project:
   cd submission

3. Install dependencies:
   pip install -r requirements.txt

4. Jalankan dashboard:
   streamlit run dashboard/dashboard.py
